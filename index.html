PipCamp

    A HipCamp-inspired Fallout-styled clone made with Flask / Python, React / Javascript, Heroku hosting, AWS image hosting

  • Detailed Flask forms allow for info-dense strings from React to be validated as complete and stored with SQLAlchemy
  • Designed Locations database objects to condense a form with 23 fields into an object with 8 columns
  • Achieved pixel-perfect recreation of inspiring web app, with the application of a different theme and color palette

TableOpen

    An OpenTable clone; a Flask / React / Redux project with 3 teammates, constructed in 4 days with 1 week of Python experience

  • Backend and Frontend components constructed in 2.5 days, all styling achieved in 1.5 days
  • Showcases the ability to combine disparate programming languages into a two CRUD feature application
  • Took care that we ensured a refined UX/UI while under time constraints

Moetaku

    A Medium clone for creating and reviewing gaming articles: Express.js, PUG, Heroku hosting.

  • First fullstack project as a team

  • Dynamically updating features for a smoothe user experience

Bark!

    A Yelp clone for places you can take your dog: Express.js, React.js, Heroku hosting

  • Utilized the Redux store to add and remove from State for dynamically available data
  • First Fullstack project as a solo developer: Place creation and Reviews/ratings on those places
  • PostgreSQL database managed with the Sequelize object-relational mapper
